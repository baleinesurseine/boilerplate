# Git

### configure for Github
```
git remote add origin https://github.com/baleinesurseine/repo.git
```
### push to Github
```
git push -u origin master
```
### pull from Github
```
git pull
```
### add file to local git
```
git add file
```
### update local git
```
git add -A
```
### commit pending updates
```
git commit -a -m 'commit message'
```
